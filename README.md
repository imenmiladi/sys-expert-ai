# **Système Expert : Détection de Pannes de Véhicules**

## **Description**

**Ce projet implémente un système expert en Python** pour détecter les pannes de véhicules à partir des symptômes observés. Il utilise :

- **Chaînage avant** : déduit automatiquement les diagnostics à partir des faits observés  
- **Chaînage arrière** : vérifie si un diagnostic particulier est déductible à partir des faits  
- **Interface graphique ergonomique avec Tkinter**, permettant de :  
  - **Sélectionner les symptômes**  
  - **Choisir le type de chaînage** (avant ou arrière)  
  - **Visualiser les résultats dans une zone scrollable**

---

## **Fonctionnalités**

### **Chaînage avant**
- **Sélectionnez les symptômes observés**  
- **Le système déduit les diagnostics possibles et affiche les faits finaux**

### **Chaînage arrière**
- **Sélectionnez les symptômes observés**  
- **Choisissez un diagnostic à vérifier dans une liste déroulante**  
- **Le système confirme ou infirme si le diagnostic est possible avec les faits**

### **Interface ergonomique**
- **Couleurs douces et design moderne**  
- **Cadres clairs pour symptômes et diagnostics**  
- **Zone scrollable pour tous les résultats**

### **Réinitialisation facile**
- **Bouton pour tout décocher et recommencer un diagnostic**
